## 🎚️ 슬라이더 조절 팁

### 볼륨 (Volume)
- **10-30%**: 배경 소음으로 은은하게
- **40-60%**: 일반적인 수면/집중용
- **70-100%**: 주변 소음 차단용

### 간격 (Interval)
- **0.1-0.4초**: 빗소리처럼 빠른 느낌
- **0.5-1.0초**: 규칙적인 물방울
- **1.5-2.0초**: 느긋한 명상용

### 피치 (Pitch)
- **-5.0 ~ -2.0**: 깊고 무거운 소리 (동굴, 지하)
- **-1.0 ~ +1.0**: 자연스러운 물방울
- **+2.0 ~ +5.0**: 맑고 높은 소리 (종소리 느낌)

### 자연스러움 (Variation)
- **0%**: 완벽히 일정 (집중용, 리듬)
- **10-20%**: 약간의 변화 (자연스러움)
- **30-40%**: 상당한 변화 (리얼한 자연)
- **50%**: 극단적 변화 (완전 랜덤)

## 💡 실험 아이디어

### 실험 1: 극단적 설정
- 모든 Variation을 50%로 설정
- 간격을 0.2초로 설정
- 어떤 소리가 나는지 들어보기

### 실험 2: 완벽한 규칙성
- 모든 Variation을 0%로 설정
- 간격을 정확히 1.0초로
- 메트로놈처럼 정확한 소리 체험

### 실험 3: 필터 비교
- 동일한 설정으로 5가지 필터 순차 적용
- WaterDrop → Basement → Cave → Pipe → Sink
- 어떤 차이가 있는지 귀로 비교

### 실험 4: 피치의 마법
- 간격 1.0초, 모든 Variation 0%로 고정
- 피치만 -5.0에서 +5.0까지 변경
- 같은 음원이 완전히 다른 악기처럼 들림!

## 📊 변동폭(Variation)의 실제 효과

### 간격 변동 30% 예시
기본 간격 1.0초일 때:
```
실제 재생: 0.85초 → 1.12초 → 0.73초 → 1.27초 → 0.94초 → ...
```

### 볼륨 변동 40% 예시
기본 볼륨 0.5일 때:
```
실제 볼륨: 0.42 → 0.63 → 0.35 → 0.58 → 0.71 → ...
```

### 피치 변동 50% 예시
기본 피치 0.0일 때:
```
실제 피치: -8 → +5 → -11 → +9 → -3 → ... (세미톤)
```
